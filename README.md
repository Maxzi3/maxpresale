## Features
-   **Dynamic Countdown Timer**: Implemented with React state, this feature displays a live countdown to the presale end, creating urgency and excitement.
-   **Interactive Presale Dashboard**: Presents critical presale statistics such as soft cap, hard cap, token price, and sold percentage, dynamically updated to reflect real-time progress.
-   **Token Purchase Module**: Facilitates a seamless user experience for connecting Web3 wallets and acquiring $MAX tokens, complete with real-time conversion rates (USDT to $MAX).
-   **Step-by-Step Buying Guide**: Offers clear and concise instructions on how to participate in the presale, enhanced with intuitive iconography for ease of understanding.
-   **Comprehensive Tokenomics Overview**: Provides a visual representation of token distribution via a responsive Pie Chart (powered by Recharts) and a detailed breakdown of allocations.
-   **Strategic Roadmap**: Outlines the project's vision, key milestones, and future development phases (Q1-Q4 2026), demonstrating a clear path for community and ecosystem growth.
-   **Dynamic FAQ Section**: An expandable accordion component that efficiently answers frequently asked questions, improving user experience and information accessibility.
-   **Responsive & Animated UI**: Designed with Tailwind CSS for a mobile-first approach and augmented with Framer Motion for smooth, engaging animations, delivering a modern and captivating Web3 aesthetic.
-   **Contract Address Display**: Features an easily accessible and copyable smart contract address, promoting transparency and enabling user verification.

## Getting Started
To get this project up and running locally, follow these steps:

### Installation
-   **Clone the repository**:
    ```bash
    git clone https://github.com/Maxzi3/maxpresale.git
    cd presale
    ```
-   **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```
-   **Run the development server**:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

## Usage
Once the development server is running, open your browser and navigate to `http://localhost:3000`.

-   The landing page will display all the current presale information, including the countdown timer.
-   Interact with the "Connect Wallet" button to link your preferred Web3 wallet and participate in the token purchase.
-   Explore the various sections like "Tokenomics," "Roadmap," and "FAQ" to gain a deeper understanding of the $MAX Token project.
